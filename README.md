# lanGame2D
A game to learn languages

Level 1
------------------
In this level 
you have to move
the circle to the 
correct translation


level 2
------------------
In the first level
you have to choose 
the right translation
between 2 different translations


level 3 (on development for next version)
------------------
In this level 
you have to write 
the correct transltion
before the word is gone
